# Smart Bathroom Automation System (IoT)

## 🚀 Overview
An IoT-based automation system designed to control and monitor bathroom utilities using embedded hardware and a mobile application.

## ⚙️ Features
- Automated lighting control using sensors  
- Real-time monitoring and control  
- Wireless communication via MQTT  
- Cross-platform mobile app (Flutter - iOS & Android)  

## 🧠 Technologies Used
- ESP32 / Microcontroller  
- MQTT Protocol  
- Embedded C  
- Flutter (Mobile App)  

## 📱 Mobile Application
Built using Flutter for both iOS and Android platforms to enable remote control and monitoring.

## 🎯 Objective
To develop a scalable IoT automation solution integrating hardware, firmware, and mobile software.
